# Processing & Preservation Web Project

This project addresses the critical issue of food processing and preservation, focusing on its importance for food security, waste reduction, safety, nutrition, and public health.

## Overview

Food processing and preservation are essential components of modern food systems. They play a vital role in transforming raw agricultural outputs into safe, nutritious, and stable food products. This project aims to educate the public about the significance of these practices and their impact on society.

## Project Structure

- **src/**: Contains the main source files for the website.
  - **index.html**: The entry point of the website.
  - **styles/**: Contains CSS files for styling the website.
  - **js/**: Contains JavaScript files for functionality.
    - **components/**: Contains reusable components like header and footer.
  - **pages/**: Contains individual HTML pages focusing on various aspects of food processing and preservation.
  - **data/**: Contains JSON data related to initiatives in food processing and preservation.

- **docs/**: Contains documentation files, including research findings and implementation guides.

- **scripts/**: Contains scripts for starting the development server.

- **tests/**: Contains tests for ensuring website accessibility.

- **package.json**: Configuration file for npm, listing dependencies and scripts.

- **.gitignore**: Specifies files to be ignored by version control.

- **LICENSE**: Licensing information for the project.

## Setup Instructions

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Install the necessary dependencies using npm:
   ```
   npm install
   ```
4. Start the development server:
   ```
   ./scripts/start.sh
   ```
5. Open your web browser and go to `http://localhost:3000` to view the website.
